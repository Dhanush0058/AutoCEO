# AutoCEO

AutoCEO is a full-stack app with a React frontend and a FastAPI backend for AI-assisted business operations.

## Structure

- frontend: Vite + React + TypeScript
- backend: FastAPI API routes for finance, HR, legal, GTM, and orchestrator flows

## Getting Started

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
